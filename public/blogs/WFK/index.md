# 雾枫枯小站

这是一个功能完整的前后端分离网站，包含题库测试、收藏夹管理、留言系统等功能。

## 功能特性

1. 用户认证系统（注册、登录、验证码登录）
2. 题库测试系统（科目分类、题库管理、在线测试）
3. 收藏夹管理
4. 留言系统
5. 主题切换（明亮/暗夜模式）

## 技术栈

- 前端：HTML5, CSS3, JavaScript, Fetch API
- 后端：Node.js, Express.js
- 数据库：MySQL
- 其他：JWT认证, CORS跨域处理


## 批量导入题目功能说明

管理员可以通过CSV文件批量导入选择题到题库中。

### 使用步骤

1. 登录管理员账户
2. 进入"题库管理"页面
3. 在"题库管理"标签页中找到"批量导入题目"区域
4. 选择目标题库
5. 下载并参考 `templates/选择题模板.csv` 文件格式
6. 准备好题目数据的CSV文件
7. 选择文件并点击"确认导入"

### CSV文件格式要求

CSV文件必须包含以下列标题：
- 题目内容
- 选项A
- 选项B
- 选项C
- 选项D
- 正确答案
- 解析

### 注意事项

1. 正确答案列只能填写 A, B, C, D 中的一个字母
2. 文件编码建议使用 UTF-8
3. 只支持选择题的批量导入
4. 导入过程中请勿刷新页面
